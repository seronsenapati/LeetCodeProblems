# LeetCode Problems

My collection of LeetCode solutions. Automatic folder organization and README updates.

## How it works:
1. Push a file named `{Number}_{title_slug}.{ext}` (e.g., `1_two_sum.py`) to the root of the repo.
2. GitHub Actions will automatically:
   - Move the file to the correct folder (e.g., `0001-1000/1_two_sum.py`).
   - Update the README.md table with the problem link and solution link.
